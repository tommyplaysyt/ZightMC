    SERVER FEATURES

    Our server has many features including: DeathChests, Custom Enchantments, Season Pass Rewards, Daily Login Rewards and many more to discover! Find out more below!

    SERVER FEATURES

    DEATH CHESTS

    A chest will spawn in the spot where you died with all of your items in it! This helps to not lose your items! You'll be given the coordinates of this chest to help find it. You can also use /back to return to the spot where you died instantly. The DeathChest Expires after 10 minutes, so be sure to collect it!

    SEASON PASS

    Unlock exclusive rewards by completing different challenges! These rewards can be SUPER OP, so we recommend doing some challenges for ingame money and Season Pass XP.

    DAILY ITEM SHOP

    An Item Shop that resets daily! Contains Custom Enchants, key items and other things too. There is planned to be many more items added to the Item Shop Rotation, so keep an eye out.
